2048

Summary

This is a recreation of the classic game 2048. This project was mainly to experiment with manipulating 2D arrays. There is the ability to use this on mobile with the addition of the arrow buttons on the bottom. This also makes use of local storage to save your high score.

A nice addition to this project was the background in the menu. This only makes random moves but it is essentially simulating games constantly and shows how the strategy of random moves does not work very well.

The notable difference between this and the original game is firstly the addition of 1 which doesn't make any difference to the actual gameplay. The other difference is the way tiles combine, it is not that noticeable but you can have chains of combinations when tiles are moving which wouldn't happen in the actual game originally this was a bug but it felt like a nice small addition to have to make the gameplay slightly different.
