Naughts and Crosses

Summary

This is a recreation of the game of naughts and crosses in JavaScript using the p5 library. There is both a 1 and 2 player mode which can be selected in the menu. The background of the menu is a simulation of a game between 2 players making completely random moves and it still correctly tracks who is winning those games.

The 2-player mode allows 2 players to take turns inputting with the mouse. Each symbol placed is placed with an animation that somewhat draws out that shape. There is also the option to choose whether the person who goes first is alternating or not. The final feature is the score between the players which is kept in the bottom right and gets reset whenever the user goes back to the main menu.

The 1 player mode allows 1 player to play against a bot. This contains the same animation that the 2-player mode does as well as the ability to choose whether the bot goes first or second. This bot is made using the min-max algorithm. Unlike Connect 4 (also featured on this site), this game is simple enough to be able to be fully solved in real-time. After each user moves the bot will go to the maximum depth to make the most optimal next move. Hence this game is impossible to win against the AI who will always force a draw or a win if a win is possible.
