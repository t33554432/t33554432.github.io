Sorting

Summary

This is a fairly common sorting algorithm visualiser. This was made in JavaScript and can visualise how each of the different algorithms works. This shows 7 different sorting algorithms that vary in speed.

The UI for this is fairly simple. It shows the number of comparisons that are counted as it goes along. The algorithm is all done in one frame when the start button is pressed and hence the time taken can be measured from that, after that it shows what the array looked like at different parts of the sort depending on the draw frequency. The number of frames to draw is calculated since the number of comparisons can be precalculated approximately which I did by making each algorithm run multiple times on each array size and then taking an average.

The options for the sort type, array size, draw delay and draw frequency can all be adjusted using the small arrows on the sides of the relevant option. The options are all limited to reasonable values. The start button will complete the sort and then run the visualisation of it.
